# Future_ML_03
Resume / Candidate Screening

This project is a  resume screening system that automatically ranks job candidates based on how well their resumes matches a given job description. It uses Natural Language Processing (NLP) and Machine Learning techniques including TF-IDF vectorization and Cosine Similarity.

 Dataset

Source: Resume Dataset — Kaggle (Sneha Anbhawal)
**LINK** : https://www.kaggle.com/datasets/snehaanbhawal/resume-dataset


Objectives

Preprocess and clean raw resume text
Extract relevant skills using keyword matching
Convert text to numerical vectors using TF-IDF
Rank candidates by their similarity to a job description using Cosine Similarity
Visualize match scores and skill gaps
Export ranked results to CSV
